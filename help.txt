# Flutter Riverpod Template Setup

## Clone Repository
```sh
 git clone https://github.com/ishworpanta100/flutter_riverpod_template
```
<button onclick="navigator.clipboard.writeText('git clone https://github.com/ishworpanta100/flutter_riverpod_template')">Copy</button>

## Activate Mason CLI
```sh
 dart pub global activate mason_cli
```
<button onclick="navigator.clipboard.writeText('dart pub global activate mason_cli')">Copy</button>

## Get Mason from GitHub
```sh
 mason get
```
<button onclick="navigator.clipboard.writeText('mason get')">Copy</button>

## List Local Mason Templates
```sh
 mason -ls
```
<button onclick="navigator.clipboard.writeText('mason -ls')">Copy</button>

## Create and Update Newly Created Flutter App
```sh
 mason make new_flutter_app -o /path/to/new_created_flutter_app/
```
